# Feature: 홈 / Todo

> 🔲 상세 기능 정의 예정

## 개요

홈 화면은 초기에는 Todo / 일정 관리 중심으로 시작한다.
추후 대시보드 화면을 별도로 추가할 예정이며, 그 시점에 메뉴 구조도 재편할 수 있다.

---

## 방향성 (확정)

- 모바일 홈 = Todo / 일정 관리 중심
- PC 홈 = 추후 전체 자산 대시보드로 확장 예정
- 나중에 홈(대시보드)과 일정관리를 별도 메뉴로 분리할 수 있음
