# Feature: 자산 현황

> 🔲 상세 기능 정의 예정

## 개요

적금 / 주식 / 부동산 등 보유 자산 현황과 수입 / 지출을 월별로 입력하여 자산의 변동성을 확인하는 기능.

---

## 방향성 (확정)

- 가계부(건건 입력)가 아닌 **월별 스냅샷** 방식으로 관리
- 자산 항목: 현금/예적금, 주식, 부동산 등
- 수입 / 지출도 월별로 카테고리 단위로 입력
- 월별 총자산 변화 추이 확인
