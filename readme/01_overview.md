# 1. 프로젝트 개요

## 1.1 배경 및 목적

개인이 직접 수기로 관리하던 엑셀 기반의 자산 현황 및 부동산 호가 데이터를 체계적으로 관리하고 시각화하기 위한 **개인용 풀스택 웹 애플리케이션**이다.

**핵심 Pain Point**
- 부동산 호가 데이터는 시간이 지나면 어디서도 찾을 수 없음 → 직접 매월 수집/기록 필요
- 엑셀로 관리하던 자산 현황 및 투자 분석 데이터를 시각화하여 인사이트 도출
- 모바일에서 현장 데이터 입력, PC에서 분석/비교를 목적으로 한 이원화된 UX

---

## 1.2 앱 이름

**Folio** — Portfolio의 줄임말로, 자산 포트폴리오를 관리한다는 의미

---

## 1.3 메뉴 구조

| 메뉴 | 모바일 역할 | PC 역할 | 상태 |
|------|------------|---------|------|
| 홈 (Todo) | 할일 체크 / 일정 관리 | 추후 대시보드 추가 예정 | 1차 개발 대상 |
| 부동산 트래킹 | 빠른 호가/실거래가 입력 | 분석, 차트, 단지 비교 | 1차 개발 대상 |
| 자산 현황 | 월별 자산/수입/지출 입력 | 추이 그래프, 항목별 분석 | 추후 개발 |
| 설정 | 공통 메타데이터 관리 | 공통 메타데이터 관리 | 추후 개발 |

> 홈은 초기에는 Todo/일정 관리 중심으로 시작하고, 추후 대시보드 화면을 별도로 추가할 예정이다.
